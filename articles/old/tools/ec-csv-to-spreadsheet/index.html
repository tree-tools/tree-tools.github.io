【GASで簡単自動化】ECサイトのCSVをスプレッドシートに自動転記するGASツール

この記事で紹介するツールについて

今回ご紹介するのは、「ECサイトから出力されたCSVデータをGoogleスプレッドシートの購入リストに自動転記するツール」です。
日々の業務で繰り返し行われるデータ転記作業を効率化するために作成しました。

1. ECサイトCSVデータ転記ツールを作った背景

ECサイトで注文データを確認し、それをスプレッドシートに手作業で転記する業務がありました。
しかしこの作業には、いくつかの課題がありました。毎回コピペ作業が発生するため手間がかかり、コピーや貼り付けの際にミスが起きやすい上、行の並び替えも頻繁に発生します。
さらに、業務を他の人に引き継ぐ際には手順の説明が複雑で、習得にも時間がかかっていました。
こうした悩みを解消するために、「誰でも正確に、短時間で作業ができるツール」を作ることにしました。

2. CSV転記ツールの概要とGASでの自動化手順

ツールの処理フロー

  1. 指定のシートにCSVの内容を貼り付ける
  2. 「転記」ボタンをクリック
  3. CSVの列を入れ替えて、既存リストの最下部に転記
  4. 転記したデータに「更新日時」「更新者名」を自動で追記

自動転記の動きと使い方

ECサイトから取得したCSVデータを、Googleスプレッドシートの指定エリアに貼り付けてボタンを押すだけ。これだけで、購入リストの末尾に自動転記され、更新者名や日付も自動で追加されます。加えて、更新日時と更新者情報も自動で付加されます。
このようにすることで、単純作業を減らしつつ、ヒューマンエラーを防ぐことができます。手順は非常にシンプルなので、初めての人でも迷うことがありません。

使用している主要なGAS関数

  • SpreadsheetApp.getActiveSpreadsheet()
    実行中のスプレッドシートを取得
  • getRange(row, column, numRows, numColumns)
    行列の番号で範囲を指定
  • setValues(array)
    2次元配列を範囲に書き込む
  • setValue(value)
    単一セルに値を設定
  • getLastRow()
    データの入った最後の行番号

Google Apps Script の基本についてはこちらの記事でも解説しています。

3. GAS開発時の工夫ポイントとミス防止策

  • CSVを貼る場所さえ間違えなければOK
  • 複雑な操作を排除し、転記の手間を最小化
  • 既に転記済みのデータかを判別して、二重登録を防止
  • 他の人に作業を引き継ぎやすいようなガイド不要のシンプル設計

開発にあたっては「誰でも簡単に使えること」を最優先にしました。

4. 導入効果

このツールを導入したことで、作業時間とミスの両方が大きく改善しました。

  • 行の並び替えなどで毎回15分程度かかっていた作業が、1分以下に短縮
  • 転記ミスや貼り間違いがゼロに
  • 作業手順が明確化されたことで、他の人にもすぐ教えられるように
  • 手間のかかるルーティン作業が、ワンクリックで完結するようになりました。

5. 応用アイデア

この仕組みは他にも以下のような業務に応用できます：

  • 在庫リストや発送リストへの転記
  • 定型フォーマットへのデータ変換
  • 各種報告用資料の自動更新

6. おわりに

日常的に発生するデータ転記作業は、地味ながら大きな時間ロスやミスの原因になります。
今回のようなシンプルな自動化ツールでも、十分な効果が得られることがわかりました。
「自動化」と聞くと難しそうに感じるかもしれませんが、実際には“貼って押すだけ”の簡単操作です。
